# Build Icons

Place your application icon files here:

- `icon.ico` - Windows icon file (required for Windows build)
- `icon.png` - PNG icon (256x256 recommended)
- `icon.icns` - macOS icon file (for macOS build)

You can use online tools to convert images to these formats:
- https://convertico.com/ - PNG to ICO
- https://cloudconvert.com/png-to-icns - PNG to ICNS

For now, the build will use a default Electron icon.
